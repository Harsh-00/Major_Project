
## Set-Up

1. Install Yarn
```
npm install --global yarn
```

2. Install Dependencies:
```
yarn install
```

## To Run on Localhost:
1. 
```
cd frontend
```
2. 
```
yarn run dev
```

### Other Commands

1. Compile Contracts:
```
yarn hardhat compile
```

2. Run Scripts:
```
yarn hardhat run <script name>
```
3. Run tests:
```
yarn hardhat test
```
